DESCRIPTION

This program fits a sequence of avalanches to the Random Field Ising Model and to the Branching Process according to the fitting procedure described in our paper.


Results of the systematic analysis of time series generated by several different systems are reported in

https://arxiv.org/abs/2109.00116

Please acknowledge the use of this software by citing the paper above.

Details regarding how to use the code and how to handle input and output are given in the notebook "weibo_full_analysis.ipynb".
